# ✨ Features List — Lucky Cruise Voyage

---

## Public Website

### Pages
- **Homepage** — Hero slider, stats, quick booking, featured cruises, why us, testimonials, promos, CTA
- **Cruises** — Filterable cruise list with search by tour type & subcategory
- **Cruise Detail** — Image gallery, overview/itinerary/inclusions tabs, seat plan, packages, reviews
- **Packages** — Tour package listings
- **Gallery** — Photo gallery with lightbox zoom
- **Blog** — Blog listing with detail pages
- **About** — Company info, team, certificates
- **Contact** — Form + Google Maps embed + WhatsApp redirect
- **404** — Custom not found page

### Features
- 🌐 Bilingual (English / Bengali) with toggle
- 📱 Fully responsive (mobile/tablet/desktop)
- 💬 WhatsApp floating button on all pages
- 🎫 Booking modal with WhatsApp integration
- ⭐ Google-style review system
- 🖼️ Image zoom/lightbox
- 🗺️ Seat plan viewer with zoom
- 🎬 Framer Motion page animations
- 🔍 SEO optimized (meta tags, semantic HTML, OG tags)
